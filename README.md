# Vimeo demo
web design of Vimeo 
## Fetures:
* mobile web design
* Desktop web design
* responsive web design


## Version
 nodejs:12.16.1

## Installation

First, clone this repo.
Second, cd vimeo_dev

Then 
`npm install`


`Accept  to install them  as  dev`

`npm start`

The website will be available at `localhost:8080`
